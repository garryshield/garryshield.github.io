查看 SELinux 状态及如何关闭

SELinux Status And How To Disable It

Tags:
SELinux
Categories:
Linux
By Garryshield, Created at 2018-08-07, Updated at 2018-08-07

1. 查看 SELinux 状态：

SELinux 有三个状态 Enforcing，Permissive，Disabled，当中 Enforcing 是预设的模式。

2. 关闭

2.1. 临时关闭

setenforce 指令可以即时切换 Enforcing 及 Permissive 这两个模式，但这些改动在系统重新开机时不会被保留。

2.2. 永久关闭

直接修改配置文件 /etc/selinux/config 禁用 SELinux。
